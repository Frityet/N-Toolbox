# N-Toolbox

An H3VR mod that adds a bunch of wrist-menu buttons:

```cs
public readonly List<string> WristMenuButtons = new List<string>()
{
	"Gather Items",
	"Reset Traps",
	"Freeze Guns",
	"Unfreeze Guns",
	"Freeze Ammo",
	"Freeze Attachments",
	"Ammo Panel",
	"Restore Full",
	"Restore 10%",
	"Toggle 1-hit",
	"Toggle God Mode",
	"Kill yourself",
	"Toggle Invisibility",
	"Add token",
	"SP - Ammo Reloader",
	"Kill patrols",
}
```

## Requirements
- Deli 0.3+ (https://github.com/Deli-Collective/Deli/releases)
- Deli.H3VR (https://github.com/Deli-Collective/Deli.H3VR/releases)

## Installation
Place the `.Deli` file into your `H3VR/Deli/mods/` folder.
